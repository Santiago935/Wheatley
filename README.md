# Wheatley
Codigo del autobot para el proyecto UNSLAM

Lo unico que se debe configurar manualmente desde el codigo es tu wifi con la contraseña. Ambas cosas estan indicadas en que parte
Puedes cambiar la velocidad del auto.

Cuando el auto se prenda, se conectara a la red. Para controlar el auto, desde arduino saldra por pantalla un http://<ip>, debes ingresar a esa direccion desde el buscador de internet. Allí aparecera por pantalla un menú con los botones para manejar el auto.
